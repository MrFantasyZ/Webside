图标文件说明
=============

插件需要以下三个图标文件：

1. icon16.png  - 16x16 像素
2. icon48.png  - 48x48 像素
3. icon128.png - 128x128 像素

您可以：

方法一：创建自己的图标
- 使用任何图片编辑软件（如 Photoshop、GIMP、Canva）
- 创建上述三个尺寸的 PNG 图标
- 建议使用紫色或蓝色调（与插件 UI 主题一致）
- 可以包含 "VIP" 字样或皇冠图标

方法二：临时使用占位图标
- 在线工具：https://www.favicon-generator.org/
- 上传任意图片，它会自动生成多个尺寸
- 下载后重命名为 icon16.png、icon48.png、icon128.png

方法三：使用纯色占位
- 暂时可以使用任意 PNG 图片，只要尺寸正确即可
- Chrome 会自动缩放显示

将三个图标文件放在 VIP 目录下即可。

注意：没有图标文件，插件仍可正常工作，只是浏览器中不显示图标。
